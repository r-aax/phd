\paragraph{Личный вклад} \

Все выносимые на защиту результаты получены автором лично.
Из 27 основных публикаций по теме диссертации 10 работ [1, 3-5, 8, 12, 16, 25-27] выполнены без соавторов.
В основных публикациях по теме диссертации автору принадлежат: метод перестроения поверхностной сетки с помощью градиентного спуска в двумерном случае [20], метод окрестностей для решения задач вычислительной геометрии в двумерном случае [1, 12, 25], метод окрестностей перестроения поверхностной неструктурированной сетки в трехмерном случае [2, 8, 10], методы удаления самопересечений поверхностной неструктурированной сетки в трехмерном случае [10, 11], метод погруженной границы с использованием фиктивных ячеек в трехмерной постановке [9, 16], архитектура объемной блочно-структурированной сетки [26], алгоритмы распределения блоков объемной блочно-структурированной сетки между вычислительными процессами с дроблением блоков [21, 27], методы декомпозиции поверхностной неструктурированной сетки [4, 14, 15], метод сглаживания границ между доменами при декомпозиции поверхностной неструктурированной сетки [13], реализация алгоритма реберной раскраски дуального графа поверхностной неструктурированной сетки [6], реализация векторизованных операций с малоразмерными матрицами для использования их в газодинамическом решателе [23, 24], методы векторизации плоских циклов на программном контексте различного вида [3, 5, 17, 19, 22], анализ эффективности векторизации плоских циклов [7, 18].
